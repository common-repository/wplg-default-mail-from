== Description ==

*WPLG Default Mail From* allows you to change the default wordpress@example.com email address that WordPress sends, and the name of the sender that the email is from. It's as simple as that.

For reporting bugs, getting support or feature suggestion, please click the link below:
http://www.wpletsgo.com/wplg-default-mail/

== Installation ==

This section describes how to install the plugin and get it working.

1. Download the archive and expand it.
2. Upload the *wplg-default-mail-from* folder into your *wp-content/plugins/* directory
3. In your *WordPress Administration Area*, go to the *Plugins* page and click *Activate* for *WPLG Default Mail From*

Once you have *WPLG Default Mail From* installed and activated you can change it's settings in *Options > WPLG Default Mail From*.
